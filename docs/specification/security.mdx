---
title: Security
description: Security considerations for the Fide ID method.
---

Fide ID is an identification method, not a proof-of-control method.

- A Fide ID does not prove control, authorship, ownership, or authority.
- The [Reference Fingerprint](/id/specification/syntax#reference-fingerprint) is a calculated identifier component derived from the [Reference Identifier](/id/specification/syntax#reference-identifier), not a security feature.
- Resolution of a Fide ID only shows method-conformant output. It does not establish trust in the identified entity or in claims made about that entity.
- Fide ID does not define method-level revocation, recovery, or deactivation signals.
