---
title: Overview
description: Overview of the did:fide method specification.
---

The `did:fide` method defines how Fide IDs conform to the DID model. Fide IDs give every entity a stable, portable identifier.

Fide ID depends on the [Fide Vocabulary](/vocabulary) for the type codes and entity-specific rules used in every identifier.

---

This specification defines:

- [Syntax](/id/specification/syntax) for the Fide ID format, parts, and core rules
- [Resolution](/id/specification/resolution) for deterministic DID resolution
- [DID Document](/id/specification/did-document) for the returned JSON DID Document shape
- [Operations](/id/specification/operations) for supported and unsupported method operations
- [Security](/id/specification/security) and [Privacy](/id/specification/privacy) considerations for method use
